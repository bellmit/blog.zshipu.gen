# 多子布局

在开发中，我们经常需要将多个Widget放在一起进行布局，比如水平方向、垂直方向排列，甚至有时候需要他们进行层叠，比如图片上面放一段文字等；  
这个时候我们需要使用多子布局组件（Multi-child layout widgets）  
比较常用的多子布局组件是Row、Column、Stack
